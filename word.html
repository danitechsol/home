INTRODUCTION.

WHAT IS WORD PROCESSING?

                                                                      
Word processing is the creation and manipulation of electronic documents using specialized software designed for this purpose. Word processing software allows users to create, edit, format, and print text-based documents such as letters, reports, memos, and other types of written communications.

Word processing software typically provides a variety of tools and features to help users create and format documents efficiently, including spell checkers, grammar checkers, text formatting tools (e.g., bold, italic, underline), alignment and indentation tools, table creation tools, and various other formatting options.

Word processing has largely replaced traditional typewriting methods, providing faster, more accurate, and more flexible ways to create and manage written documents.
                                                                      

EXAMPLES OF WORD PROCESSING APPLICATIONS

                                                                      
There are several popular word processing applications available today. Some examples of commonly used word processing software include:

1. Microsoft Word: This is one of the most popular and widely used word processing applications, available as part of the Microsoft Office suite. It offers a wide range of features and tools for creating and formatting documents.

2. Google Docs: This is a web-based word processing application that is part of the Google Workspace suite of productivity tools. It allows multiple users to collaborate on a document in real-time.

3. Apple Pages: This is a word processing application available on Apple devices, including Mac, iPhone, and iPad. It offers a range of templates and design tools for creating visually appealing documents.

4. LibreOffice Writer: This is a free, open-source word processing application that offers similar features to Microsoft Word.

5. WPS Office Writer: This is another free word processing application that offers a range of features for creating and formatting documents.

These are just a few examples of the many word processing applications available today, each with its own set of features and benefits.
